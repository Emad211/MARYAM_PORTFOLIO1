# **App Name**: LinguaSage

## Core Features:

- Mission Display: Display the educator's mission statement in a visually appealing manner.
- Qualification Showcase: Showcase the educator's qualifications, such as a TestDaF certificate.
- Blog Display: Display blog posts and filter by category.
- Multilingual Support: Multilingual support for Persian, German, and English content.
- Dark Mode: Dark mode theme using inverted contrast with the same color palette.
- Homepage Hero Section: Typographic hero section with a key slogan and artistic profile picture.
- Educator's Manifesto: A short section to express the teaching philosophy.
- Call to Action Section: A minimal and powerful section to guide users to the classes page or free course registration.
- Blog Selection: Display of 3 recent articles in an asymmetrical and attractive layout.
- About Page Content: A combination of text and image to narrate the educator's story and background.
- Interactive Timeline: A graphic and interactive timeline to display the professional path.
- Classes Presentation: Presentation of various courses and classes (private, group, workshops).
- Dynamic Timetable: Dynamic timetable with the ability to filter based on class type and level.
- Class Details Page: Inner page for each class type with full descriptions, course objectives, and prerequisites.
- Registration Form: Integrated registration form with payment gateway (if needed in the future) and confirmation email sending.
- Article Listing Page: Article listing page with filtering by category.
- Full-text Article Search: Full-text search capability among articles.
- Article Page Design: Article page with editorial design, focus on readability, use of large images and prominent quotes.
- Related Articles Section: "Related Articles" section at the end of each post.
- Contact Form: A minimal and user-friendly contact form.
- Contact Information Display: Contact information and social media links graphically.

## Style Guidelines:

- Primary color: Burnt Terracotta (#B85C38) for buttons and interactive elements, evoking warmth and an academic feel.
- Background color: Parchment Cream (#F5F0E8), a warm, desaturated hue for a sophisticated, non-stark backdrop.
- Accent color: Dark Olive Green (#5A6650) to provide subtle contrast and highlight specific details.
- Headline font: 'Playfair Display' (serif) for headings to give an editorial, high-end feel.
- Body font: 'Inter' (sans-serif) for body text to ensure modern legibility and contrast.
- Asymmetrical layout to break the monotony, create visual interest and guide the user's eye.
- Meaningful whitespace to put focus on the content and creating a sense of luxury and calm.
- Artistic placement of images and text blocks with slight overlapping to create depth and visual dynamics.
- Subtle and purposeful animations using Framer Motion library.
- Parallax effects while scrolling to create depth.
- Soft and gradual appearance of content (fade-in/slide-up) when entering the user's view.
- Creative hover effects on links and images.
